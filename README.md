# FARM Stack App

A simple full-stack application built with the FARM stack (FastAPI, React, MongoDB).

## Purpose

This project is for **learning purposes only**. It demonstrates the basics of building a full-stack application using modern web technologies.

## Tech Stack

- **F**astAPI - Backend API framework
- **R**eact (Next.js) - Frontend framework
- **M**ongoDB - Database

## Project Structure

```
farm_stack_app/
├── backend/     # FastAPI backend
└── frontend/    # Next.js frontend
```

## Getting Started

### Backend

```bash
cd backend
uv sync
uv run main.py
```

### Frontend

```bash
cd frontend
bun install
bun dev
```
